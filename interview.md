# The Interview — Live Business Discovery

> **What this is.** The worksheet you run with the audience during the Open House. Fill in the answers live (out loud, on the projector via `Interview.html`, or right here in this file). Then drag this file + `BusinessGenerator.md` into Claude Code and say *"Build the entire company now."*
>
> **The vibe:** this is a live TV show, not a form. Keep it fast, funny, and generous. Let the audience shout answers. The wilder the business, the better the demo.

---

## How to run it (read this once before the event)

1. **Claude asks. You don't.** Frame it as: *"I'm going to let the AI interview one of you."* Read the questions as if the AI is asking them. It makes the machine feel like it understands business.
2. **Pick a volunteer with a real business** ("Who here owns a business?") — or invent one live with the room.
3. **Move fast.** ~30–45 seconds per question. Momentum is the magic. Don't over-explain.
4. **Encourage bold answers.** The example answers below are deliberately varied — some ordinary, some absurd — to give people permission to be creative. Read a couple aloud if the room is shy.
5. **Save the name for the end.** You name the company *together*, after everyone can already picture it. That's the emotional peak of the interview.
6. **Capture the answers** in the blank worksheet at the bottom (or click "Copy answers" in `Interview.html`), then hand it to Claude Code.

There are no wrong answers. Every blank you leave, Claude Code fills with a smart, realistic guess.
